# Notes App
A simple notes app :) 

# Getting started

## Installation

```
npm install
or
yarn install 
```
## Requirements

This app uses [expo](https://expo.dev/)

```
npm install --global expo-cli
```

## Running app

```
yarn start // Opens Expo CLI
yarn android // Opens Android emulator (if there is any) 
yarn ios // Opens IOS emulator (if there's any)
```
